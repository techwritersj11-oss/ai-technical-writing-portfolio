## Sample Excerpt

### Create a New Bot

1. Log in to the Conversational AI platform.
2. Navigate to **Bots → Create New Bot**.
3. Select the bot type:
   - Chatbot
   - Voice Bot
4. Enter basic details such as bot name and language.
5. Click **Create**.

Once created, the bot is ready for intent configuration and channel integration.

---

### Configure Intents

Intents define what the user wants to achieve.

To configure intents:
1. Go to **NLU → Intents**.
2. Click **Add Intent**.
3. Provide training phrases that reflect user queries.
4. Save and test the intent using the simulator.
